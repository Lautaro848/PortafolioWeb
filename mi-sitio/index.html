Estás abriendo la página desde el sistema de archivos (file://). Algunos navegadores bloquean recursos o módulos cuando se usa file://. Para ver la mascota 3D correctamente, sirve la carpeta con un servidor local. En PowerShell ejecuta: 
py -m http.server 8000
Después abre http://localhost:8000/mi-sitio/
